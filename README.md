# Flights app: React + Spring Boot + Redis (Docker)

Este proyecto contiene:

- 📦 **Frontend**: React + Vite (Port 5173)
- 🔧 **Backend**: Spring Boot con Gradle (Port 8080)
- 🧠 **Redis**: Cache
Everything run with `docker-compose`.
